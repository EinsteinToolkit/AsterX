Standalone version of the RIT_EOS thorn. The purpose here is to test the EOS
table interpolation routines.

Compile by typing "make" in in the current directory and run with "./Main".

********************************************************************************
N.B.: you have to provide a valid EOS in Stellarcollapse format before compiling
********************************************************************************

You can edit the "user-defined parameters" in Main.cc ; then, recompile and
re-execute.
